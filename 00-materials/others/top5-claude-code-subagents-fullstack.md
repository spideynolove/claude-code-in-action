# Top 5 Claude Code Subagent Projects for Fullstack AI Dev Teams

Ranked by GitHub stars (as of March 2026).

---

## 1. affaan-m/everything-claude-code — 110k stars

**URL:** https://github.com/affaan-m/everything-claude-code

**Description:** "The agent harness performance optimization system" — a complete system of skills, instincts, memory optimization, continuous learning, security scanning, and research-first development. Anthropic Hackathon Winner.

**Tech Stack:**
- Languages: TypeScript, Python, Go, Java, Shell, Perl, Rust
- Backend: NestJS patterns, Spring Boot, Django, Laravel, Axum
- Frontend: React, Next.js
- Infra: Docker, PM2, Railway, Vercel, CI/CD
- DBs: PostgreSQL, Supabase, ClickHouse
- Testing: Playwright, pytest

**Highlights:** 28 specialized agents, 125+ skills, 60+ slash commands. Framework configs for Django, FastAPI, NestJS, Go (Axum patterns). Covers the full dev lifecycle: code review, test automation, build resolution. Multi-tier agent model using Opus/Sonnet/Haiku strategically.

## 2. hesreallyhim/awesome-claude-code — 32.7k stars

**URL:** https://github.com/hesreallyhim/awesome-claude-code

**Description:** Canonical curated meta-list of skills, agents, plugins, hooks, slash-commands, agent orchestrators, and tools for Claude Code.

**Tech Stack:**
- Languages: Python, Rust, Go, TypeScript, Node.js, Bash
- Frameworks: React, Vue 3, Docker
- Includes: Ruflo (multi-agent swarm orchestration), Parry (prompt injection scanner)

**Highlights:** The discovery hub for the Claude Code ecosystem. Contains the Fullstack Dev Skills collection (65 specialized skills). If a team needs the right agent/skill/plugin for FastAPI, NestJS, Golang, or mobile, this is the starting index.

---

## 3. wshobson/agents — 32.3k stars

**URL:** https://github.com/wshobson/agents

**Description:** Intelligent automation and multi-agent orchestration for Claude Code — 112 agents, 72 plugins, 146 skills, 79 dev tools across 24 categories.

**Tech Stack:**
- Languages: Python, JavaScript/TypeScript, Go, Java, Rust, Kotlin, C#
- Backend: FastAPI, Django, NestJS, Express, Spring Boot
- Mobile: React Native, Flutter
- Infra: Kubernetes, Terraform, Docker, AWS/Azure/GCP
- ML/AI: LangGraph, RAG, vector DBs

**Highlights:** Most comprehensive fullstack-oriented multi-agent system. Three-tier model strategy (Opus/Sonnet/Haiku per task complexity). Agent Teams plugin for parallel workflows. Conductor plugin for context-driven development. Explicitly covers FastAPI, NestJS, Go, React Native, and Flutter — hits every criterion.

**Best all-around pick for a fullstack team starting from scratch.**

## 4. VoltAgent/awesome-claude-code-subagents — 15.3k stars

**URL:** https://github.com/VoltAgent/awesome-claude-code-subagents

**Description:** 127+ specialized Claude Code subagents across 10 categories covering the full development lifecycle.

**Tech Stack:**
- Mobile: React Native, Flutter, Expo, Swift
- Backend: FastAPI, Go/Golang, NestJS, Spring Boot, Django, Laravel, Rails
- Frontend: React 18+, Vue 3, Angular 15+, Next.js 14+
- Infra: Docker, Kubernetes, Terraform, AWS/GCP/Azure

**Highlights:** Dedicated `fullstack-developer.md` subagent and a `nestjs-expert.md` agent. Deep mobile coverage (React Native + Flutter + Expo + Swift). 430 commits, actively maintained. Ideal for teams that want drop-in subagents per tech domain.

## 5. VoltAgent/awesome-agent-skills — 13k stars

**URL:** https://github.com/VoltAgent/awesome-agent-skills

**Description:** 1,030+ official Agent Skills from real engineering teams at Anthropic, Google, Vercel, Stripe, Cloudflare, Netlify, Supabase, HashiCorp, and 40+ others. Compatible with Claude Code, Codex, Gemini CLI, Cursor, Copilot, and OpenCode.

**Tech Stack:**
- Frontend: React, React Native, Next.js, Tailwind, shadcn/ui
- Backend: Node.js/TypeScript, Python, Golang
- Infra: Cloudflare Workers, Netlify, Vercel, Terraform
- DBs: PostgreSQL, DuckDB, ClickHouse, Neon
- Specialized: Remotion, Replicate, Firecrawl

**Highlights:** Official-team-backed skills (not community bulk-generated). Every skill comes from the actual engineering team of that service. Golang backend coverage, React Native for mobile, multi-agent-tool compatible.
