# Debug

1. Restate the observed symptom in one sentence
2. List the possible root causes, most likely first
3. For each cause, identify the minimal evidence that would confirm or rule it out
4. Check the most likely cause first — read the relevant file or run the relevant command
5. Once confirmed, propose the fix with the exact change needed
6. If not confirmed, move to the next candidate

Be direct. No speculation without evidence. Show the fix as code, not prose.
