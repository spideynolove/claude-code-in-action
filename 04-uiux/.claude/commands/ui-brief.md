Create or update an `IntentSpec` for the current task.

Arguments: $ARGUMENTS

## Your job

Convert the request into a structured product/UI intent document.

## Must include

- product type
- primary user
- main jobs to be done
- risk level
- interaction density
- tone
- key constraints
- avoid_styles
- preferred_styles
- accessibility expectations
- responsive expectations

## Rules

- focus on user goals before visuals
- infer missing but reasonable constraints
- keep it implementation-aware
- do not generate components or code here
- save the result as structured JSON or update the current intent file
